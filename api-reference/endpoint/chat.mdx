## MMK Chat
Use this endpoint to generate conversational, multi-turn chat responses using large language models available on the Omni Model Market.  
This endpoint supports system prompts, user messages, assistant context, and optional streaming.

---

## Your First API Call
All chat completions are done through a single endpoint:

```bash
POST /v1/mmk-chat
```

---

## Authentication
Requires API Key authentication. Include your API Key in the `Authorization` header:

```bash
Authorization: Bearer <your-api-key-here>
```

---

## Request

### Headers

| Header          | Type   | Required | Description                    |
| --------------- | ------ | -------- | ------------------------------ |
| `Authorization` | string | Yes      | Bearer token with your API key |
| `Content-Type`  | string | Yes      | Must be `application/json`     |

---

### Request Body

| Field              | Type    | Required | Description |
| ------------------ | ------- | -------- | ----------- |
| `model`            | string  | Yes      | Chat model ID (e.g. `openai/gpt-5`) |
| `messages`         | array   | Yes      | List of chat messages with roles (`system`, `user`, `assistant`) |
| `verbosity`        | string  | No       | Response detail level (`low`, `medium`, `high`) |
| `reasoning_effort` | string  | No       | Controls reasoning depth (`minimal`, `medium`, `high`) |
| `image_input`      | array   | No       | Optional image inputs for multimodal chat |
| `stream`           | boolean | No       | Enable streaming responses |

---

### Message Object

| Field   | Type   | Required | Description |
| ------ | ------ | -------- | ----------- |
| `role` | string | Yes      | `system`, `user`, or `assistant` |
| `content` | string | Yes | Message text |

## Example Request

### Python Example (Streaming)

```python
import requests

url = "https://modelmarket.allcognix.com/v1/mmk-chat"

payload = {
    "model": "openai/gpt-5",
    "messages": [
        {
            "role": "user",
            "content": "Tell me the story of the green goblin in 50 words."
        }
    ],
    "verbosity": "medium",
    "reasoning_effort": "minimal",
    "stream": True
}

headers = {
    "Authorization": f"Bearer {API_KEY}",
    "Content-Type": "application/json"
}

with requests.post(url, json=payload, headers=headers, stream=True, timeout=120) as response:
    for line in response.iter_lines():
        if line:
            print(line.decode(), end="", flush=True)
```

---

### cURL Example

```bash
curl -X POST "https://modelmarket.allcognix.com/v1/mmk-chat" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
        "model": "openai/gpt-5",
        "messages": [
          {
            "role": "user",
            "content": "Tell me the story of the green goblin in 50 words."
          }
        ],
        "verbosity": "medium",
        "reasoning_effort": "minimal",
        "stream": false
      }'
```
