## Web Interface

The simplest way to experience Omni Model Market is through our intuitive Web UI.

<Steps>
  <Step title="Create an Account">
    Visit <a href="https://modelmarket.allcognix.com" target="_blank">modelmarket.allcognix.com</a> and sign up to access the dashboard and playground.
  </Step>
  <Step title="Get Your API Key">
    Open your Profile → **API Keys** → Generate your personal API key. This key is required for all API requests.
  </Step>
  <Step title="Test Models in Playground">
    Go to **/playground** to try text, image, video, audio, and code models with one unified interface.
  </Step>
  <Step title="Monitor Usage">
    View credits, cost breakdowns, and request history in real time from your dashboard.
  </Step>
</Steps>

### Web Interface Features

- **Unified Playground** for all model types  
- **Model listing** with top models and task based 
- **Usage & cost tracking**  
- **API Key management**  
- **Multi-modal generation** (text, image, video, audio, code)

## API Integration

For all the developers who want a fully programmable flows to utilise most of it.

<Steps>
  <Step title="Create Your API Key">
    Sign in → Profile → API Keys → Create API Key.
  </Step>

  <Step title="Send Your First API Request">
    Use our generation or streaming endpoint to run a model.
  </Step>

  <Step title="Handle Responses">
    Each response includes output, model type, cost, and status.
  </Step>

  <Step title="Monitor your tasks">
    Frequently visit the web dashboard to observe your usage, cost, time spent, and other fields.
  </Step>
</Steps>

### Key Concepts

**Models List**: Choose the best suitable model as per your requirement and give it a try in the web playground to assess the output as per your input.

**Models API**  
Endpoint like generate and chat for these models: text, image, video, audio, and code.

**Cost Tracking**  
Every request returns exact credit usage. Notification system will inform automatically for low balance remainings.

**Payment**: Pay as you go method will help you to quickly decide the needful amount to recharge without worrying to cancel or upgrade.

# API Example
### Endpoint 

POST https://modelmarket.allcognix.com/v1/mmk-generate

```bash
curl -X POST "https://modelmarket.allcognix.com/v1/mmk-generate" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
        "model": "minimax/speech-02-hd",
        "input": "Voice cloning technology that offers voice synthesis"
      }'
```
