## Run Any Model Directly in the Browser

Omni Model Market web interface provides an intuitive way to experiment and use models without writing any code. Perfect for non-technical users, content seekers, and somebody who wants to quickly set up and test featured models capabilities.

## Accessing the Web Interface

Visit [modelmarket.allcognix.com](https://modelmarket.allcognix.com) to access our web interface. You'll need to create an account if you don't have one already.

## Key Features

### Intuitive Explore Section

**Explore the models**
* Browse through the plenty of models on [modelmarket.allcognix.com/explore](https://modelmarket.allcognix.com/explore)
* Select the best model for your fit and navigate to the playground.
* We also have "featured" page for the popular models in the ongoing period, sorted with task and model type.

### Perfect Playground for you
**Run the model**
* Once you have selected the model, navigate yourself to the playground for the chosen model.
* Fill the fields with inputs (one can ignore the default fields if don't want to tweak much) and hit the run button.
* Wait few seconds/minutes for the model to run and see the result in the seperate section.
* You will be getting the basic details like output content, cost and time elapsed for the model. One you can see in the below example picture:

### Dashboard in detail

**Navigate to Profile**

* Go to your profile and dashboard section
* You can see all the details of the recent tasks which you have created either by web playground or by API calls.
* Track your progress status, cost, timestamp, input and output details. Refer the attached image of dashboard section:

### Visual + Downloadable Results

After running a model:
* Images/videos/audio can be previewed directly
* Outputs are shown instantly
* All files can be downloaded in one click
* API request & response JSON is visible for developers

**Query Features**

* Ask follow-up questions
* Get summaries of specific topics
* Find specific information quickly

### Great for Prototyping

With our intuitive playground you can quickly test:
- Marketing content
- TTS voices
- Design variations
- Small video clips
- AI assistants or chat features
- Before integrating further into your app or pipeline.

**Cost Management**

* See real-time cost tracking
* You can choose the amount to recharge as per your need.
* Monitor your usage against low and zero balance limits
* Get notifications for payment and balance limits.
